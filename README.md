# Classical Information Retrieval Models

While reading paper about the probabilistic foundations of BM25, I became interested in implementing some of the earlier ranking models to better understand how BM25 evolved.
I would try to implement some classical information retrieval (IR) weighting models that could be considered as predecessors of modern BM25 and compare their retrieveal and behavior performance
